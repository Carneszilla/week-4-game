How to Play

Each image button has a random value assigned to them at the start of a new game

  1. CLICK ON ACTUAL PICTURES TO START A GAME
  2. This will generate a random number
  3. You use these numbers to match the number in the Points to Match space
  4. If you go over the, Points to Match number, you lose.
  5. You need to match the Points to Match exactly to win
  6. CLICK ON ACTUAL PICTURES TO PLAY
  7. Every new game started all numbers are goin to be differen, everything changes.
